    📖 單詞闖關遊戲規則

    🎯 遊戲目標：
    挑戰 20 個關卡，通過答對題目來獲得星星和積分！

    📋 基本規則

    • 每關有 10 道題目
    • 需要達到 60% 正確率（答對 6 題）才能通關
    • 通關後可解鎖下一關
    • 必須按順序通關，不能跳關

    ⏱️ 時間限制

    • 選擇題：30 秒作答時間
    • 填空題：40 秒作答時間（因為需要打字）
    • 超時未作答視為錯誤，會顯示正確答案
    • 答錯或超時後，會顯示 8 秒讓您學習正確答案

    📝 題目類型

    • 🎧 聽力選擇：聽英文發音，選擇正確的中文意思
    • 📖 閱讀翻譯：看英文單詞，選擇正確的中文意思
    • ✏️ 填空題：根據中文提示，拼寫出英文單詞

    ⭐ 評分系統

    • ⭐⭐⭐ (3星)：正確率 ≥ 90%
    • ⭐⭐ (2星)：正確率 ≥ 70%
    • ⭐ (1星)：正確率 ≥ 60%
    • 可以重複挑戰已通關的關卡來提高星星數

    🎁 積分獎勵

    • 完美通關：必須答對全部 10 題（100% 正確率）才能獲得積分
    • 每天完成闖關可獲得 最多 2 積分
    • 每完美通關一關即可獲得 1 積分
    • 積分會計入您的總積分和成長樹
    • 注意：如果有答錯題目，即使通關也不會獲得積分

    💡 小提示

    • 題目中的單詞可以點擊 🔊 按鈕聽發音
    • 選項也有 🔊 按鈕，可以聽粵語或英語發音
    • 填空題提供中文提示和英文發音提示
    • 答對後會隨機顯示粵語表揚語句鼓勵您！

    ⚡ 快捷添加單詞

    批量導入單詞數據

    📋 使用說明：
    • 粘貼 JSON 陣列格式的單詞數據
    • 支援批量添加多個單詞
    • 格式示例見下方說明

    🤖 AI 生成提示詞：
    "請幫我將以下生詞轉換為 JSON 格式，用於單詞學習應用：
    要求：每個單詞必須包含english和customTranslation(粵語翻譯)。
    如果有常用例句，可以添加 examples 陣列，每個例句包含 en（英文例句）和 zh（中文粵語翻譯）。例句要求簡單點，並且是香港小學生可以理解的"。
    輸出格式必須是 JSON 陣列，不要其他文字。"

    JSON 格式示例：
    [
      {
        "english": "apple",
        "customTranslation": "蘋果",
        "examples": [
          {"en": "I eat an apple.", "zh": "我食蘋果。"}
        ]
      }
    ]

    📝 支援的格式：

    基本格式（需要 english + chinese）：
    {"english": "apple", "chinese": "蘋果"}

    使用自定義翻譯（需要 english + customTranslation）：
    {"english": "run", "customTranslation": "跑步；經營"}

    完整格式（可選字段）：
    {
      "english": "apple",
      "chinese": "蘋果",
      "customTranslation": "自定義翻譯",
      "examples": [{"en": "I eat an apple.", "zh": "我吃蘋果。"}]
    }
